Git 常用命令

2017-11-7

  • 初始化一个版本库

    git init

  • 克隆远程版本库

    git clone <url>

  • 添加远程版本库

    git remote add origin <url>

  • 查看远程仓库

    git remote -v

  • 提交修改到暂存区，有三种模式可以选择（三选一）

    // 提交所有变化
git add -A
// 提交被修改(modified)和被删除(deleted)文件，不包括新文件(new)
git add -u
// 提交新文件(new)和被修改(modified)文件，不包括被删除(deleted)文件
git add .

  • 提交修改

    git commit -m "提交说明"

  • 推送到远程仓库

    git push origin master

  • 查看文件状态

    git status

  • 跟踪指定文件

    git add readme.txt

  • 从当前跟踪列表移除文件，并删除本地文件

    git rm readme.txt

  • 仅从暂存区删除，仍保留本地文件

    git rm --cached readme.txt

  • 重命名文件

    git mv reademe.txt readme.html

  • 查看提交历史

    git log

  • 修改最后一次提交的提交注释

    git commit --amend

  • 创建一个分支

    git branch <name>

  • 切换工作目录

    git chekcout <name>

  • 删除分支

    git branch -d <name>

  • 查看所有分支（有*号的为当前分支）

    git branch

  • 查看远程仓库的信息

    git remote show origin

  • 强制掩盖远程分支

     git push origin <name> --force
