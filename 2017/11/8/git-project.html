Git https、http 方式设置记住用户名及密码

2017-11-8

Git 使用 https、http 方式时，每次都要手动输入密码，按照如下方法设置后，只需输入一次密码，以后再不用承受手动输入密码的困扰。

设置记住密码（默认15分钟）：

git config --global credential.helper cache

自定义密码失效时间（一小时之后失效）：

git config credential.helper 'cache --timeout=3600'

长期存储密码（推荐一）：

git config --global credential.helper store`

小提示：增加远程地址时加上用户名和密码也是可以的（推荐二）

git remote add origin http://username:password@git.oschina.net/name/project.git

补充：使用客户端也可以存储密码。
